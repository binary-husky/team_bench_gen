问题：Z3 经常把位向量（bit-vector, BV）约束通过「位爆破（bit-blasting）」求解——把每个 n 位向量展开为 n 个布尔变量、把每个 BV 运算展开为一个布尔电路，再交给 SAT 引擎。请解释：为什么这一归约到 SAT 对【固定位宽】位向量既是可靠的（sound）又是完备的（complete）？进而说明，为什么同样的方法对【无界】整数算术不适用——是什么根本差异使得位爆破对 BV 正确且有限、对无界整数却不行？

---

[Judge]

Look at `./summary_logic.md`, check whether conclusion cover the following points

1. 标准答案：固定位宽 n 位的位向量恰有 2^n 个取值，且每个 BV 运算（加、减、乘、移位、有/无符号比较）都是一个固定的组合布尔电路（如逐位进位加法器），其布尔语义与该 BV 运算的模 2^n 语义完全一致；故一个 BV 公式可满足，当且仅当其位爆破后的布尔电路可满足——可靠且完备，且编码有限（每向量 n 位）。而无界整数算术没有固定位宽、整数可任意大，不存在有限的 n 可爆破；位爆破会需要无穷多位、无法在有限 SAT 实例上终止。故无界整数算术必须用专门的算术理论求解器（如线性/Presburger 算术判定过程），而非位爆破到 SAT。

1. 须给固定位宽 n 位 BV 恰 2^n 取值、每 BV 运算为固定组合布尔电路（模 2^n 语义逐位一致）⇒ BV 公式可满足 ⟺ 位爆破布尔电路可满足（可靠且完备、编码有限每向量 n 位）；而无界整数无固定位宽、不可有限爆破、须专用算术理论求解器（如 Presburger）。（细化原 [Judge] 第 1 点）
