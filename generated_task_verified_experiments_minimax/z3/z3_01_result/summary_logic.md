# 为什么位爆破对固定位宽位向量既可靠又完备，但对无界整数不适用

> 任务来源: `task.md` —— 读给定材料,仅通过逻辑推理回答。
> 材料: de Moura & Bjørner, *Z3: An Efficient SMT Solver* (2008),尤其是 §3 "System Architecture" 中
> "Theory Solvers" 一节的明确陈述:
> *"The fixed-sized bit-vector theory applies **bit-blasting** to all bit-vector operations, but equality."*
>
> 下面的全部推导基于这份材料给出的事实,加上命题逻辑、SAT 决策性,以及一阶结构/模型论
> 的标准结论,没有任何实验测量。

---

## 1. 什么是位爆破(bit-blasting)

设所有 BV 变量都已声明位宽 `n_i`。位爆破做的事是:

1. 把每个 `n` 位 BV 变量 `x` 替换为 `n` 个布尔变量 `x_{n-1}, …, x_0`(通常下标 0 是最低位)。
2. 把每个 BV 操作符 `op` 替换为它在该位宽上的**布尔电路**(由 `∧ / ∨ / ¬ / xor` 等门组成的
   与门电路,等价于操作符的真值表 / 位级定义)。例如:
   - `bvadd`(无符号/二进制补码加法) → 行波进位加法器电路
   - `bvmul` → shift-and-add 电路
   - `bvand / bvor / bvxor` → 逐位的 `∧ / ∨ / xor`
   - `bvshl / bvlshr / bvashr` → 移位 + 0/1 填充
   - 比较运算 `bvult` 等 → 最高位比较电路
   - 抽取/拼接 `extract / concat` → 直接按位重连
3. 把所有这些电路 Tseitin 编码成 CNF,交给 DPLL(T) 内部的 SAT 求解器。

整个过程没有任何近似:电路就是该 BV 操作在 `ℤ_{2^n}` 上的定义本身,只是换了表示层
(从整数值换到位向量)。

---

## 2. 为什么对【固定位宽】BV 既可靠(sound)又完备(complete)

### 2.1 可靠(sound):"SAT 给出 sat ⇒ BV 公式可满足"的方向

设位爆破把 BV 公式 `Φ` 翻译成 CNF 公式 `Φ_B`。我们必须证明:

> 若 SAT 求解器返回 `sat` 并给出赋值 `α`,则 `α` 提升回 BV 后是 `Φ` 的一个模型。

理由是**逐位同构**(bitwise isomorphism)。把 `α` 中对应 BV 变量 `x` 的那些
布尔变量 `{x_{n-1}, …, x_0}` 收集起来,把它们按照 BV 的语义
(下标 0 为最低位的 little-endian 解释,或根据 signedness 解释)拼回一个 `n` 位向量值。
得到一个把每个 BV 变量映射到 `ℤ_{2^n}` 中元素的赋值 `M`。

对任意一个 BV 子项 `t` 归纳证明:

> 在 `α` 下,`t` 的电路输出第 `i` 位取真值 ⟺ 按 BV 语义 `[[t]]_M` 的第 `i` 位为 1。

基础:变量 — 由构造,`α(x_i)` 就是 `M(x)` 的第 `i` 位。
归纳步:加法/乘法/移位/抽取/拼接等每一种 BV 操作,它的布尔电路恰好是
该操作在位级上的**定义**——电路的每根输出线都按照与硬件 RTL 相同的真值表
工作。所以电路输出逐位等于按 BV 语义计算的值的逐位。

由此,任何形如 `t1 = t2`、`t1 ≤_u t2`、`bvult(t1,t2)`、`extract(t,i,j)=…` 的 BV 原子,
电路把它编码为一个布尔变量(或一组布尔变量),而该布尔变量在 `α` 下为真
**当且仅当**该原子在 `M` 下为真。谓词符号(由 E-matching 等机制实例化出的整
体约束)同理,要么来自 `=` 之外的 BV 原子(都被翻译),要么是已经在 BV 层定义好
的常量和模式。

所以 `α ⊨ Φ_B` ⇒ `M ⊨ Φ`。任何被 SAT 接受的赋值都能被反向解码成一个真正的
BV 模型。**没有假阳性** ⇒ 可靠。

(等号本身:`=` 不位爆破、而是交给 §3 的 congruence closure core 处理。Equivalence
类的合并在 E-graph 上传播,而 E-graph 的节点本身就是 BV 项(操作结果),它们的等价
关系恰好对应于 BV 层的"两个 BV 项必须逐位相等"。所以即便等号走另一条路,语义也
等价于按位比较。)

### 2.2 完备(complete):"BV 公式可满足 ⇒ SAT 给出 sat"的方向

设 `Φ` 在 `ℤ_{2^{n_1}} × ℤ_{2^{n_2}} × …` 上有模型 `M`。把 `M(x)` 的逐位拆出来
得到一个布尔赋值 `α`(对每个 BV 变量 `x` 的每一位给一个真值)。同样按上面的逐位
同构论证:

- BV 操作按位等价于它的布尔电路,所以每个 BV 子项 `t` 的电路在 `α` 下输出
  的位串就是 `[[t]]_M` 的位串。
- 任何 BV 原子在 `M` 下为真 ⟺ 电路编码出的布尔变量在 `α` 下为真。
- `M ⊨ Φ` ⇒ `α ⊨ Φ_B`。

所以 SAT 求解器至少存在一个满足赋值,它一定能找到(或等价地证明存在)。**没
有假阴性** ⇒ 完备。

### 2.3 终结性 / 决策性 — 为什么这套归约是"有限"的

位爆破的输入中:

- BV 变量的个数是有限的;
- 每个 BV 变量的位宽 `n_i` 是**固定的、有限**的(这正是材料里"fixed-sized
  bit-vector"措辞的含义,也正是 Z3 把它和 linear arithmetic 分到不同 theory
  solver 的原因);
- 每个 BV 操作符对应**一个固定的、有限**的布尔电路(加法是 `n` 个全加器,
  乘法是 `n` 个 `n` 位 shift-and-add 阵列,等等);
- 因此 CNF 中变量数 = `Σ n_i`,子句数 = `O(Σ n_i · circuit_size)`,
  都是有限自然数。

命题可满足性 (SAT) 本身是 NP 完全但**可判定的**:对任何有限 CNF,DPLL(T) 必
在有限时间内返回 `sat` 或 `unsat`。所以固定位宽的 BV 理论是可判定的——位爆
破不仅 sound 且 complete,还**有界终止**,正好对应一个决策过程 (decision
procedure)。

### 2.4 把三点合在一起

| 性质 | 来自哪里 |
|------|----------|
| 可靠 (sound) | 布尔电路是 BV 操作的逐位定义本身,反向解码给出真模型 |
| 完备 (complete) | 任何 BV 模型按位拆开后是 CNF 的一个赋值,只要存在就能找到 |
| 终止 / 决策性 | 输入有限(位宽固定 ⇒ 变量数、电路规模有限) ⇒ SAT 必终止 |

这三件事缺一不可,而位爆破对固定位宽 BV 三件都满足,所以它是 Z3 在这种特定理
论下的"杀手锏"。

## 3. 为什么同样的位爆破对【无界】整数算术不适用

关键差异是**底层一阶结构的基数**:

- 一个 `n` 位 BV 变量取值范围是 `ℤ_{2^n}`,**有限集**;
- 一个无界整数变量取值范围是 `ℤ`,**可数无限集**。

这个区别看似简单,后果是根本性的。

### 3.1 没有任何有限命题编码能忠实地表示 `ℤ`

命题逻辑(SAT 的语言)只能表达有限多个布尔变量的真值赋值。其表达能力有
一个**基数上界**:用 `k` 个布尔变量最多区分 `2^k` 个值。这是组合数学的事实,
没有任何巧妙的编码可以绕过。

如果想用有限个布尔变量 `b_1, …, b_k` 表示一个整数 `x ∈ ℤ`,把布尔赋值映
射到 `ℤ` 的函数 `f: {0,1}^k → ℤ` 一定是**多对一**的——像样的定义域 `ℤ` 是
无穷的,值域 `{0,1}^k` 只有 `2^k` 个元素。`f` 把无穷多个整数映射到同一个布尔
赋值上。

后果:**任何 `f` 都不可能区分"足够大"的整数**。具体地,假设 `f` 把 `b = 000…0`
映射到 `v_0`,那么 `v_0 + 2^k` 这个整数没有任何一组布尔赋值能把它和 `v_0` 区分
开——所有 `k` 位布尔串都用完了。

反过来,这意味着任何"用 `k` 位编码无界整数"的尝试都会有**饱和点
(saturation point)**:对所有绝对值大于某个 `M` 的整数,编码后的位模式开始折
叠。约束语言只要能写出 `x > M`(在 ℤ 上显然合法),就能写出一条约束,它在饱和
之前为真、之后为假、然后又为真,而任何 `k` 位编码都会"忘记"自己在哪个区域。

所以:固定位宽的 BV 本质上**就是 `ℤ_{2^n}`**——有限结构;而所谓"无界整数"是
`ℤ`——无限结构,二者根本不是同一回事。无界整数不是"位宽很大的 BV",因为那个
"位宽"必须是有限的,而把一个有限位宽当作"无界"会改变理论的语义。

### 3.2 没有有限电路能实现 `ℤ` 上的算术

对固定位宽,`bvadd / bvmul / bvshl` 等都有有限的逐位电路。对 `ℤ` 上的加法
和乘法:

- 加法(`x + y`):结果需要 |x| + |y| + 1 量级的位才能无溢出地表示。在最坏情况
  下没有固定的位宽——任意 `n` 位电路都会对足够大的输入溢出。这不是电路"实现
  得不够好",而是 ℤ 上的 `+` 本身没有有限真值表:对任意 `n`,都存在 `x, y`
  使 `x + y` 的二进制表示超过 `n` 位,而布尔电路在 `n` 位输出线上无法区分
  `2^n + 5` 和 `5`。
- 乘法(`x · y`):更糟,需要 |x|·|y| 量级的位。同样没有有限电路。
- 除法和模运算:本身也是"无界结果"的运算。

换句话说,即便我们假装"取一个超大的位宽 `N`",也只在问题里出现的整数都小
于 `2^{N-1}` 时是对的;而 BV 输入规范里写下的位宽就是这个 `N`,问题里所有
BV 变量的取值范围都被预先约束在 `[-2^{N-1}, 2^{N-1})`。**位爆破的有效性不是
来自"我们对算术的实现多巧妙",而是来自"输入规范已经把整数域剪裁到了一个有
限集上"。** 无界整数没有这种剪裁。

### 3.3 算术层的可判定性也不是免费午餐

位爆破能工作的另一隐含前提是:**我们要解的理论本身就是可判定的、并且我们已经
有一个可终止的求解器去逼近它**。对 ℤ 上的纯线性整数算术(QF_LIA,Presburger
算术,只含 `+`、常数、整数变量、不含 `·`),Matiyasevich / Cooper / Dijkstra
等人的工作表明它是可判定的,只是代价昂贵(在最坏情况下是指数级乃至三重指数级
时间和空间)。Z3 因此需要专门的 theory solver (§3 提到基于 Yices 算法的
*"The arithmetic theory ... is based on the algorithm used in Yices"*)——不
能用 SAT。

如果再乘以非线性乘法 `·`,得到 QF_NIA,**甚至不可判定**(Matiyasevich 定理,
Hilbert 第十问题的否定回答)。此时根本不存在决策过程,自然也不存在"先归约到
SAT 再求解"这条路。

无论可判定与否,ℤ 算术的**难度本质**在于操作需要随输入值大小变化的资源;SAT
是固定大小的有限搜索,无法匹配这个要求。

### 3.4 把四点合在一起

| 维度 | 固定位宽 BV | 无界整数 `ℤ` |
|------|------------|--------------|
| 值域基数 | 有限 `2^n` | 可数无限 |
| 命题可编码性 | 可,`n` 位 = `n` 个布尔变量 | 不可,任何 `k` 位编码都会饱和 |
| 是否有有限电路实现每个运算 | 有 | 无(加/乘需要随输入增长的位) |
| 理论本身 | 命题可满足,NP-完全但可判定 | LIA 可判定但代价高;非线性乘法不可判定 |
| 位爆破可行? | 是 | 否,本质不可能 |

---

## 4. 一句话总结

位爆破之所以对固定位宽 BV 同时可靠、完备且有限,是因为**固定位宽把理论剪
裁到了一个有限集 `ℤ_{2^n}` 上**——每个 BV 变量变成 `n` 个布尔变量、每个 BV
操作变成它的逐位布尔电路,这种翻译是逐位同构的,所以 SAT 的 `sat/unsat` 与
BV 公式的可满足性一一对应,且因为命题数和子句数都有限,SAT 必定终止。

位爆破对无界整数 `ℤ` 不适用,根本原因不是技术性的(不是"电路不够好"或"启发
式不够聪明"),而是**结构性的**:命题逻辑的有限基数编码不可能区分无穷多个整
数,任何固定 `k` 位编码都会在 `2^k` 处饱和,任何有限布尔电路都无法无溢出地
实现 `ℤ` 上的加法或乘法。这恰好是 de Moura & Bjørner 在 §3 中把"bit-vector
theory solver"和"linear arithmetic theory solver"**分列为两个不同模块**的根
本原因——前者可以用 SAT 位爆破,后者必须用专门的可判定性算法(Yices 风格的
simplex / cuts / branch-and-bound),而这并不是 Z3 的缺陷,而是 `ℤ` 与 `ℤ_{2^n}`
这两个一阶结构本身的差异。
