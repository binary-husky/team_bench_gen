# NSGA-II 基线复现：ZDT1 vs ZDT2 (n_var = 30)

> 课题：用固定实验设置在 ZDT1 / ZDT2 (n_var=30) 上复现标准 NSGA-II，作为后续课题的基线。

## 1. 固定实验设置（全部如实执行）

| 组件 | 设置 |
|---|---|
| 算法 | 标准 NSGA-II（二元锦标赛 + 锦标赛精选 + 精英保留） |
| 种群大小 `N` | 100 |
| 终止代数 | 250（共 250 × 100 = 25000 次适应度评估） |
| 交叉 | SBX，`eta_c = 20`，`p_c = 0.9` |
| 变异 | 多项式变异，`eta_m = 20`，`p_m = 1/n_var = 1/30` |
| 独立运行次数 | 31（seeds = 1, 2, …, 31） |
| 评价指标 | IGD（参考前沿解析采样 1000 点）、HV（参考点 (1.1, 1.1)）、IGD 每 25 代的收敛曲线 |

实现细节：使用 `pymoo==0.6.x` 的 `NSGA2`（即经典 Deb 等 2002 算法），对 SBX/PM 用任务规定的参数显式覆盖默认值；自己逐代驱动算法循环以便在第 25, 50, …, 250 代记录 IGD（任务指定的“每隔 25 代”采样）。

## 2. 最终结果（31 次独立运行的均值 ± 标准差）

| 指标 | **ZDT1** | **ZDT2** | ZDT2 / ZDT1 比 |
|---|---|---|---|
| **IGD**（越小越好） | **0.026109 ± 0.012187** | **0.066040 ± 0.083865** | 2.53× |
| **HV**（越大越好） | **0.833280 ± 0.018570** | **0.443023 ± 0.074411** | 0.53× |
| IGD min / max | 0.010743 / 0.072683 | 0.018366 / 0.498767 | — |
| HV  min / max | 0.762984 / 0.857259 | 0.098579 / 0.508935 | — |
| IGD 中位数 | 0.026447 | 0.048219 | — |
| HV  中位数 | 0.832957 | 0.461746 | — |

> 注：标准差使用样本标准差 (`ddof=1`)。ZDT2 的 IGD 最大值 0.499（seed=3）是一个明显 outlier，对均值有较大拉升；其中位数（0.048）更能代表“典型”表现，但仍明显高于 ZDT1。

观察：

- **ZDT1**：IGD 在 250 代内从 ~0.47 单调下降到 ~0.026（约 18× 改善），标准差从 0.054 下降到 0.012，曲线光滑且方差很小。
- **ZDT2**：IGD 从 ~0.96 下降到 ~0.066（约 14× 改善），下降速度明显更慢；而且标准差始终在 0.10 上下，几乎是 ZDT1 的 7–8 倍，说明**不同种子之间高度不一致**——某些运行（seed=3 的 IGD≈0.50）几乎没有逼近前沿。

## 4. 结论：ZDT2 比 ZDT1 难多少？

**ZDT2 在本基线下明显更难**，量化如下：

1. **IGD 高约 2.5×**：ZDT2 的平均 IGD（0.0660）≈ ZDT1（0.0261）的 2.53 倍；只看中位数差距略小（0.0482 / 0.0264 ≈ 1.83×），但都 > 1。
2. **HV 低约 47%**：ZDT2 平均 HV（0.4430）只有 ZDT1（0.8333）的 53%，等价于 ZDT1 的 HV 比 ZDT2 **大 88%**（(0.8333 − 0.4430) / 0.4430 ≈ 0.881）。
3. **方差显著更大**：
   - IGD 标准差：ZDT2 (0.0839) 是 ZDT1 (0.0122) 的 **约 6.9 倍**；
   - HV  标准差：ZDT2 (0.0744) 是 ZDT1 (0.0186) 的 **约 4.0 倍**。
   - 说明 NSGA-II 在 ZDT2 上对初始化更敏感，部分种子（seed=3）几乎失败。
4. **收敛更慢**：在同样的 250 代预算内，ZDT1 已经把 IGD 压到 ~0.026 且标准差很小；ZDT2 在第 250 代的 IGD 仍然几乎与 ZDT1 在第 125 代时相当（0.066 vs 0.106）。ZDT2 在 1000 代内能否追平 ZDT1 在 250 代的水平，本实验未验证（任务只到 250 代）。
5. **结构性原因**：ZDT2 的真实 Pareto 前沿是**非凸**曲线 `f2 = 1 − f1²`，而 ZDT1 是**凸**的 `f2 = 1 − √f1`。NSGA-II 基于 Pareto 秩 + 拥挤距离的选择压力，在非凸前沿上更难维持均匀分布的端点（端点 `(0,1)`、`(1,0)` 处的可行情形差异大），这是经典文献中已经记录的 ZDT2 比 ZDT1 难的主要原因之一。

## 5. 后续课题建议（不在本次任务范围）

- 若需要更可靠的 ZDT2 基线，可延长到 500–1000 代，或用 ε-支配归档 + 自适应算子。
- 可尝试在每次运行时检查是否过早收敛到前沿的某一段（特别是 (0,1) 附近），并据此分类“成功/失败”以解释 ZDT2 的高方差。
- 参考点 (1.1, 1.1) 与解析前沿完全包含区域 [0,1]×[0,1] 内，最大可能 HV ≈ 1.21（对 ZDT1）或 ≈ 0.84（对 ZDT2，因为前沿更凹）；用这个上界可换算“归一化 HV”以便跨问题比较。
