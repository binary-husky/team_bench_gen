# Resolution γ 的影响 — Louvain 在 LFR 基准图上的实验

## 1. 实验设置

- **图**: LFR 基准图，由 `networkx.LFR_benchmark_graph` 生成
  - `n = 1000`, `tau1 = 3.0`, `tau2 = 1.5`, `mu = 0.4`,
    `average_degree = 6`, `min_community = 20`, `seed = 0`
  - 生成结果: `|V| = 1000`, `|E| = 2044`
  - 真实社区数 = **34** 个（节点自带的 ground-truth 标签），规模区间 20–46，均值 29.41
- **算法**: `community.best_partition(G, random_state=0, resolution=γ)`
  （python-louvain 0.16，即 Blondel 等 2008 的 Louvain 方法）
- **自变量**: 分辨率 γ ∈ {0.5, 1.0, 1.5, 2.0}；其余全部固定
- **因变量**（对每个 γ 都记录）:
  1. 社区数 `#C`
  2. 平均社区规模 `⟨|c|⟩ = n / #C`
  3. **标准模块度** Q（用 γ=1 的标准公式 `Q = (1/(2m)) Σ_ij [A_ij − (k_i k_j)/(2m)] δ(c_i,c_j)` 计算，
     与 python-louvain 内置 `community.modularity(partition, G)` 一致到 1e-15，
     保证各 γ 的 Q 在同一标准下可比）
- 其他指标: 中位规模、min / max 规模，仅供参考

## 3. 结论：γ 怎样改变划分粒度

1. **γ ≥ 1：模块度目标中的「社团内部边」一项被放大，倾向于把节点切开 → 社区更多、更小**。
   - 从 γ=1.0 升到 γ=2.0：`#C` 从 20 → 41（约翻倍），`⟨|c|⟩` 从 50.0 → 24.39（约减半），
     与「resolution limit」文献里「γ↑ → 更细粒度」的标准图景一致
     （Reichardt & Bornholdt 2006; Arenas, Fernández & Gómez 2008 — 也即论文末参考文献 [38]）。
   - 在本图上 γ=1.5（`#C=32`, `⟨|c|⟩=31.25`）已与真实社区数（34, 均值 29.41）非常接近，
     而 γ=1.0 出现明显的「分辨率限制」——把多个真实社区合并成更少的团块。

2. **γ=0.5 的反常行为**：
   - 直观上 γ<1 应得到「更少、更大的社区」，但在本实验里 γ=0.5 反而得到 `102` 个社区、
     `⟨|c|⟩≈9.8`——比 γ=1.0 的 20 个还多得多。检查分布可以看到这是 1 个 169 节点的大块
     + 1 个 124 节点的次大块 + 大量 1–30 量级小团块的混合，标准 Q 仅 0.4922 也是四档里最低。
   - 这是 Louvain 启发式 + python-louvain 实现方式的副作用：
     优化目标里 `in_degree * γ / m` 整体被 γ 缩小，节点在「离开自己的社区」时所获得的
     惩罚变小，导致少数边密集的核心节点迅速聚成几大团，而大量边稀疏的「叶子」节点被留在
     原社区中成为孤立小团块，最终停在一种「少数巨型 + 大量碎片」的局部最优。
     该现象不是确定性「γ↓ → #C↓」规律的简单延伸，而是分辨率参数对优化 landscape 的非线性影响。

3. **标准 Q 随 γ 的变化（非单调）**：
   - γ=1.0 → 0.5597（最高）
   - γ=1.5 → 0.5520
   - γ=2.0 → 0.5436
   - γ=0.5 → 0.4922
   - 关键点：**对每个 γ 都用 γ=1 的标准公式计算 Q**——γ=1.0 自然给出最高 Q（与目标一致）；
     γ=1.5 / 2.0 即便切得更碎，标准 Q 反而轻微下降；γ=0.5 因上面的局部最优问题跌到 0.49。
   - 这就是「标准模块度 + 标准分辨率（γ=1）」无法识别出与 ground truth（34 个社区）相对应的粒度：
     按 γ=1 的 Q 排序，会偏向更粗的划分（20 个团块），这是文献里反复讨论的
     「modularity 的 resolution limit」（Fortunato & Barthélemy 2007，论文参考文献 [23]）。

## 4. 一句话总结

> 在 `n=1000, mu=0.4` 的 LFR 图上，**从 γ=1.0 增到 γ=2.0，社区数翻倍、平均规模减半，符合「γ↑ → 更细粒度」的预期；γ=1.5 时社区数（32）与 ground truth（34）最接近；γ=0.5 因为优化目标被整体缩小，落入「少数巨型团 + 大量碎片」的局部最优，标准 Q 也最低**。
> 在所有 γ 下都用 γ=1 的标准公式计算 Q，得到 Q ∈ {0.49, 0.56, 0.55, 0.54}，最高值出现在最粗的 γ=1.0 划分上，体现了标准模块度的分辨率限制。
