用 Python **进程内**从零实现 RAPPOR 的 **Permanent Randomized Response (PRR)**：对一个 Bloom 比特 `b`，`Pr[PRR=1|b=1] = 1−f/2`，`Pr[PRR=1|b=0] = f/2`（等价于：以概率 `f` 替换成均匀随机比特，否则保留）。可用 NumPy。

研究目标：**实证验证 PRR 的每比特概率比上界 ≈ (2−f)/f = e^{ε_perm}（局部差分隐私界）。**

固定实验设置（不要更改）：
- 取两个"相邻输入"群体：一组全部 `b=1`（≥ 1e5 个比特样本），一组全部 `b=0`（≥ 1e5 个）。
- 对参数网格 **f ∈ {0.1, 0.25, 0.5, 0.75, 0.9}**，分别对两组样本跑 PRR，统计经验 `Pr[PRR=1|b=1]` 与 `Pr[PRR=1|b=0]`。
- 计算经验比值 `r̂ = Pr̂[1|b=1] / Pr̂[1|b=0]`，并与理论 `(2−f)/f` 比较；换算经验 `ε̂ = ln(r̂)` vs `ε_perm = ln((2−f)/f)`。
- 每个 `f` 用 **≥ 5 个不同随机种子**重复，取均值。

需要记录/报告的指标：
- 每个 `f` 下经验比值 `r̂` 与理论 `(2−f)/f`、经验 `ε̂` 与 `ln((2−f)/f)`。
