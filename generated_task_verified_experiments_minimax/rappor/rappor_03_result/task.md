用 Python **从零实现完整 RAPPOR 编码-解码管线**（NumPy 即可，**仅 CPU、< 30 分钟**）：
- 编码：值 `v` → Bloom 向量 `B`（`k` 比特、`h` 个哈希）。
- PRR：`f` 参数（per-bit）。
- IRR：`p, q` 参数（per-bit）。
- 聚合 + 解码：收集 `N` 个客户端的 IRR 报告，对每个比特用 `(观测到的 1 比例 − p)/(q − p)` 去偏还原 Bloom 比特概率；再在**候选字符串字典**上做最小二乘 / Lasso 回归，恢复各候选的频率估计。

研究目标：**验证完整管线能从含噪 IRR 报告聚合恢复 cohort 的真实字符串频率，并量化估计误差。**

- 解码后，比较**估计频率**与**真实频率**，计算误差（L1 误差 `Σ|p̂−p|` 与最大绝对误差 `max|p̂−p|`）。
