# 误差随查询敏感度 Δf 的变化 — 实验结论

## 1. 背景与目的

读 Dwork–McSherry–Nissim–Smith 2006 (*Calibrating Noise to Sensitivity in
Private Data Analysis*, 见 `laplacedp_material/`) 可知，对任意函数
`f : D^n → ℝ^d`，若其全局 L1 敏感度

    Δf  :=  max ‖f(x) − f(x′)‖_1     (对所有仅相差一行的 x, x′)

则 Laplace 机制

    M(x) = f(x) + Y,   Y ~ Lap(Δf/ε) ⊂ ℝ

即满足 ε-indistinguishability（Proposition 1）。本实验固定 ε，对若干查询
分别加入尺度 `b = Δf/ε` 的 Laplace 噪声，重复多次，记录**平均绝对误差**
(MAE)，用以考察「误差随查询敏感度 Δf 如何变化」。

由于 `Y ~ Lap(0, b)` 的 E|Y| = b，理论上：

    E[|M(x) − f(x)|] = b = Δf / ε                  (★)

即 MAE 关于 Δf 严格**线性**、与 n 无关。

## 2. 实验设置（全部固定，仅查询类型变化）

| 参数            | 取值        | 说明                       |
|-----------------|------------|----------------------------|
| 隐私预算 ε      | 1.0        | **固定**                    |
| 数据规模 n      | 1000       | **固定**                    |
| 试验次数 trials | 20000      | **固定**，蒙特卡洛估计       |
| 随机种子 seed   | 42         | **固定**，结果可复现         |
| 区间上界 B      | 10         | 用于求和查询                 |
| **自变量**      | **查询类型**（即 Δf） | |

查询类型（与 Dwork 等 2006 文中 Example 3 完全一致）：

| 查询                      | 形式                              | Δf     |
|--------------------------|-----------------------------------|--------|
| (a) 计数 (noisy sum)     | f(x) = Σ x_i,    x_i ∈ {0,1}      | 1      |
| (b) 有界区间求和         | f(x) = Σ x_i,    x_i ∈ [0, B]     | B = 10 |
| (c) 均值                 | f(x) = (1/n) Σ x_i, x_i ∈ [0,1]   | 1/n    |

实现见 `exp_sensitivity.py`、`exp_sensitivity_curve.py` 与
`plot_sensitivity.py`。

## 3. 三个查询类型的实验结果

```
ε=1.0, n=1000, trials=20000, seed=42

query                                                  Δf     b=Δf/ε   theory MAE   empirical MAE
------------------------------------------------------------------------------------------------
a) Count query       (binary, Δf=1)                1.0000     1.0000       1.0000           1.0053
b) Bounded sum query  (x_i ∈ [0,B], Δf=B, B=10)   10.0000    10.0000      10.0000           9.9988
c) Mean query         (x_i ∈ [0,1],   Δf=1/n)     0.0010     0.0010       0.0010           0.0010
```

empirical / theoretical 比值：1.0053, 0.9999, 0.9946 — 全部 ≈ 1（误差 < 1 %，纯蒙特卡洛抖动）。

## 4. 跨数量级的灵敏度扫描

为更直观地展示「误差随 Δf 的变化」这一函数关系，对 Δf 在
[10⁻³, 10²] 上做 16 个等比采样点，每个点同样跑 20000 次，结果（节选）：

```
        Δf       b=Δf/ε   theory MAE    empirical MAE    ratio
----------------------------------------------------------------
    0.0010       0.0010       0.0010           0.0010   1.0035
    0.0100       0.0100       0.0100           0.0099   0.9904
    0.1000       0.1000       0.1000           0.0994   0.9935
    1.0000       1.0000       1.0000           0.9957   0.9957
   10.0000      10.0000      10.0000           9.9599   0.9960
  100.0000     100.0000     100.0000          99.5741   0.9957
```

跨 5 个数量级，empirical MAE 与 `Δf/ε` 的比值始终在 [0.98, 1.01] 区间内，
与 Dwork 等给出的理论公式 (★) 完全吻合。完整 16 个点的数据见
`sensitivity_curve.json`，对数–对数坐标下的可视化见
`sensitivity_plot.png`：经验点 (圆点) 几乎完全贴在斜率为 1 的虚线
`MAE = Δf/ε` 上。

## 5. 结论

1. **线性关系**。在 ε 固定时，Laplace 机制的 MAE 与查询的全局敏感度 Δf
   **严格成正比**：MAE = Δf/ε。这是 Proposition 1 的直接推论 —
   噪声尺度 `b = Δf/ε` 而 `E|Y| = b`。

2. **数量级跨度**。把 Δf 从 1/n = 0.001 拉到 B = 10 乃至 100，MAE 也
   从 10⁻³ 涨到 10² — 跨越 5 个数量级。三个查询 (a)(b)(c) 的 Δf 比
   1 : 10 : 0.001，MAE 比 1 : 10 : 0.001，与理论一致。

3. **与 n 无关**。注意 (c) 均值查询的 Δf = 1/n 随 n 缩小 — 正是因为
   「每行贡献缩小」，求和变 1/n；而 (a)(b) 求和类查询的 Δf 与 n 无关。
   实验中保持 n = 1000 不变，仍然只在 (c) 上看到 MAE 比 (a) 小 3 个量级
   （10⁻³ vs 1），清楚地把 n 的角色剥离出来。

4. **实用启示**。差分隐私的「代价」= Δf — 任何把单条记录的影响放大
   的查询（区间求和、直方图、机器学习目标函数…）都需要相应放大的
   噪声。若要保效用，应尽量将查询**重写**为低敏感度形式：
   - 区间求和 (Δf = B) → 计数 (Δf = 1) 通过改变单位；
   - 单点求和 (Δf = B) → 均值 (Δf = 1/n) 通过除以 n；
   - 直方图逐 bin 加噪（敏感度 2）优于在整张表上 O(√d) 加噪。
   这与论文 §3.2 中 *Histograms and Disjoint Analyses* 的论述完全一致。

5. **隐私–效用权衡**。ε 固定时，敏感度与效用（MAE）线性挂钩；
   任何想用差分隐私做统计的实践者都必须先量化其查询的真实 Δf，
   再决定 ε 该取多小，否则误差可能完全压垮信号。
