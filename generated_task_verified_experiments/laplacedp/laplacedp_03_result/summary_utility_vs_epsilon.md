# ε 对实用性的影响：Laplace 机制的隐私–效用权衡

> 对应材料：Dwork, McSherry, Nissim, Smith, *Calibrating Noise to Sensitivity in
> Private Data Analysis* (2006)。该文的核心结论是：对查询函数 *f*，按其
> 全局敏感度 Δ*f* 标定噪声标准差即可实现 ε-差分隐私；Laplace 机制即将
> 服从 `Laplace(scale = Δf/ε)` 的噪声加到真值上。本实验据此量化 ε 与
> 效用（误差）之间的关系。

## 1. 实验设置（全部固定，唯一自变量为 ε）

| 项目 | 取值 | 说明 |
|---|---|---|
| 查询 | 计数查询 `count` | 数据库 N=10 000 中满足某谓词的记录数 |
| 真值 *f(D)* | 1234 | 效用指标相对真值度量，对该常数平移不变 |
| 敏感度 Δ*f* | **1** | 增加/删除一条记录，计数至多改变 1 |
| 机制 | `Laplace(0, scale=Δf/ε)` | noise = `Lap(0, b)`，输出 = 真值 + noise |
| ε 取值 | {0.01, 0.1, 0.5, 1, 2, 5} | 唯一自变量 |
| 试验次数 *T* | 200 000 | 每个 ε 独立重复 |
| 随机种子 | 0 (`np.random.default_rng(0)`) | 固定，可复现 |

度量指标：输出相对真值的**平均绝对误差 MAE = (1/T)Σ|输出−真值|**
与**均方根误差 RMSE = √((1/T)Σ(输出−真值)²)**。
由于误差 = 噪声本身，MAE/RMSE 即 `Lap(0, b)` 的样本矩。

## 2. 理论参考（Δ*f*=1）

对 `Laplace(0, b)`，*b = Δf/ε*：

- `E|X| = b` ⇒ **理论 MAE = Δf/ε**
- `E[X²] = 2b²` ⇒ **理论 RMSE = (Δf/ε)·√2**
- 标准差 `std = b·√2`，与 RMSE 相同（零均值）。

即两个误差都应**随 ε 线性反比下降**，随 *b = Δf/ε* 线性上升。

## 4. 结果与理论噪声尺度 Δf/ε 的对比

实测与理论几乎完全吻合（相对误差 < 0.4%，源自有限样本蒙特卡洛波动）：

- **MAE ≈ Δf/ε**：每一行的实测 MAE 与理论 `Δf/ε` 列逐项相等（0.01→100、
  0.1→10、1→1、5→0.2）。MAE 直接读出噪声的期望绝对值。
- **RMSE ≈ (Δf/ε)·√2 ≈ 1.414·MAE**：每行 RMSE/MAE 比值恒为 √2，符合
  Laplace 分布二阶矩 `E[X²]=2b²`。
- 因此**实测误差曲线就是理论噪声尺度曲线本身**：MAE 沿 `1/ε` 走，
  RMSE 沿 `√2/ε` 走，二者都随 ε 线性反比下降。

## 5. ε → 效用（隐私–效用权衡）解读

1. **单调反比**：误差随 ε 增大而单调下降，关系严格为 `误差 ∝ 1/ε`。
   ε 翻倍 ⇒ MAE/RMSE 减半；ε 增 10× ⇒ 误差降为 1/10。
2. **强隐私区（小 ε）误差爆炸**：ε=0.01 时 b=100，MAE≈100、RMSE≈141。
   对一个量级仅为千的计数，噪声远超信号，输出几乎不可用——这正是
   “强隐私保证代价极高”的直观体现。
3. **弱隐私区（大 ε）误差趋零但保证变弱**：ε=5 时 b=0.2，MAE≈0.2，
   计数精度很高，但 ε-差分隐私的界 `e^ε` 已达 ~148，对个体的保护显著
   退化。
4. **线性权衡的来源**：Laplace 机制把 `scale = Δf/ε` 直接写进噪声分布，
   因此效用损失与 ε 是**严格的 1/ε 反比关系**，没有饱和/平台——隐私每
   提升一个数量级（ε 小 10×），效用误差就恶化 10×，二者等比例交换。
5. **Δ*f 的杠杆**：在固定 ε 下，误差同时正比于 Δ*f*。计数查询 Δ*f*=1
   已是最小敏感度；对均值查询 Δ*f*=1/N，同等 ε 下误差会小 N 倍，这也是
   材料中“按敏感度标定噪声”带来的实用价值。

## 6. 结论

对固定计数查询（Δ*f=1）：

- MAE = Δf/ε，RMSE = (Δf/ε)·√2，二者均与 ε 成**严格反比**；
- 实测值与理论噪声尺度 Δf/ε 在全部六个 ε 上吻合（<0.4% 蒙特卡洛误差）；
- 隐私–效用权衡呈线性 `1/ε`：ε 从 0.01→5（500×）时，MAE 从 ~100→0.2、
  RMSE 从 ~141→0.28，等比例下降。强隐私（小 ε）以噪声爆炸为代价，
  弱隐私（大 ε）精度高但保护弱——不存在“免费午餐”，选择 ε 即在
  这条 `1/ε` 直线上选点。

*Δ*f=1, T=200 000, seed=0, 唯一自变量 ε。*
