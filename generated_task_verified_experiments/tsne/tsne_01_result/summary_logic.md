# 目标困惑度 > n−1 时 t-SNE 的高斯带宽与嵌入退化（纯逻辑推导）

## 结论速览

当用户把目标困惑度设成大于 `n−1` 的值时：

1. **二分搜索会把 σ_i 一路推向无穷大（σ_i → ∞）**，因为目标困惑度对 `p_{·|i}` 而言是不可达的上界。
2. 此时 `p_{·|i}` 退化为**关于其余 n−1 个点的均匀分布**：`p_{j|i} = 1/(n−1)`（∀ j≠i），`p_{i|i}=0`。
3. 对称化后的联合相似度 **P 退化为“全连接完全图”上的均匀分布**：`p_{ij} = 1/(n(n−1))`（∀ i≠j），`p_{ii}=0`，即所有点对两两等相似、毫无结构。
4. 最终的低维嵌入**坍缩成位于原点附近的单一团点**：所有 n 个点几乎重合为一点，`q_{ij}` 也趋于均匀 `1/(n(n−1))`，`KL(P‖Q)` 趋于全局最小 0，任何聚类/拓扑结构全部丢失。

下面给出完整推导。

---

## 1. 为什么困惑度有上界 n−1

在 t-SNE 中，点 i 的高维条件相似度为（见材料原文 §2，SNE 定义）

```
p_{j|i} = exp(-||x_i - x_j||² / 2σ_i²)  /  Σ_{k≠i} exp(-||x_i - x_k||² / 2σ_i²),   j ≠ i
```

注意 `p_{·|i}` 是定义在**除 i 自身外的 n−1 个点**上的离散概率分布（共 n−1 个支撑点，`p_{i|i}=0` 不计入）。

困惑度定义为 `Perp(P_i) = 2^{H(P_i)}`，其中 `H(P_i) = −Σ_{j≠i} p_{j|i} log₂ p_{j|i}` 是以 bit 计的香农熵。

离散分布的基本事实：支撑集大小为 K 的分布，其熵满足 `0 ≤ H ≤ log₂ K`，等号（上界）当且仅当分布为**均匀分布**时取得。这里 `K = n−1`，故

```
0 ≤ H(P_i) ≤ log₂(n−1)   ⟹   1 ≤ Perp(P_i) ≤ n−1.
```

因此**任何**点 i 的困惑度都被严格封顶在 `n−1`。把目标困惑度设成 `> n−1`，就等于要求一个不可能达到的熵值 `H > log₂(n−1)`。

---

## 2. σ_i 关于困惑度的单调性 → 二分搜索把 σ_i 推到 ∞

记 `f(σ_i) = Perp(P_i(σ_i))`。把 `p_{j|i}` 看成 `σ_i` 的函数：

- 当 `σ_i → 0⁺`：高斯极窄，最近的点几乎拿走全部概率质量，`p_{·|i}` 退化为单点质量，`H → 0`，`Perp → 1`（下界）。
- 当 `σ_i → ∞`：`exp(-||x_i−x_j||²/2σ_i²) → 1` 对所有 `j≠i` 一致成立，归一化后 `p_{j|i} → 1/(n−1)`，`H → log₂(n−1)`，`Perp → n−1`（上界）。
- 中间段单调：`σ_i` 越大、分布越平坦、熵越大、困惑度越大。即 `f(σ_i)` 是 `σ_i` 的**连续严格单调递增**函数，值域 `[1, n−1)`（上界是渐近线，不取到）。

二分搜索的目标是解 `f(σ_i) = 目标困惑度`。但目标 `> n−1 = sup f`，方程**无解**。二分搜索每轮比较 `f(σ_i)` 与目标：因为 `f(σ_i)` 永远小于目标，它判定“困惑度还不够大 → 需要更大的 σ_i”，于是不断抬高搜索区间的上界、压低下界试探，**σ_i 被一路向上推**。数值实现中通常有一个 σ 的上界（或对精度 β=1/2σ² 的下界），搜索会撞到这个上界；在数学极限上即

```
σ_i → ∞.
```

（实现细节：常见代码对精度 β 做二分搜索，β↓ ⟺ σ↑；为追上不可达的困惑度，β 被压向 0，σ 走向 +∞。结论一致。）

---

## 3. p_{·|i} 退化成什么

由第 2 步的极限 `σ_i → ∞`，

```
p_{j|i} = exp(0)/Σ_{k≠i} exp(0) = 1/(n−1),   ∀ j≠i.
```

即 `p_{·|i}` 退化为**其余 n−1 个点上的均匀分布**。这正是把熵推到上界 `log₂(n−1)` 的唯一分布——二分搜索虽未“命中”目标，但它竭尽全力把分布拉向了与目标方向一致的极限，即上界处的均匀分布。此时困惑度达到可达的最大值 `n−1`，但仍 `< 目标`。

---

## 4. 对称化后的联合相似度 P

t-SNE 的对称化为 `p_{ij} = (p_{j|i} + p_{i|j}) / (2n)`，`p_{ii}=0`。代入均匀条件分布：

```
p_{ij} = ( 1/(n−1) + 1/(n−1) ) / (2n) = (2/(n−1)) / (2n) = 1/(n(n−1)),   ∀ i≠j,
```

且 `p_{ii} = 0`。因此 **P 退化为**：对角元为 0、所有非对角元相等 `1/(n(n−1))` 的矩阵——也就是**完全图 K_n 上边权均匀的相似度矩阵**。几何上，这代表“任意两点都同等相似”，数据中**完全没有局部/全局结构可言**。

（即便经过早期放大 early exaggeration 乘以常数因子 α，P 仍只是被整体缩放，结构上依旧是均匀的；不改变下述结论。）

---

## 5. 低维嵌入 q_{ij} 与最终形态

低维相似度为 Student-t 核归一化：

```
q_{ij} = (1 + ||y_i − y_j||²)⁻¹ / Σ_{k≠l}(1 + ||y_k − y_l||²)⁻¹,   i≠j,  q_{ii}=0.
```

代价 `C = KL(P‖Q) = Σ_{i≠j} p_{ij} log(p_{ij}/q_{ij})`。当 P 均匀时，`C = −Σ p_{ij} log q_{ij} + const`，最小化等价于**最大化 E_P[log q_{ij}]**，即让所有 `q_{ij}` 尽量大且尽量相等。

**全局最优：完全坍缩。** 若所有点重合于一点（`y_i = y_j`，∀i,j），则每个 `||y_i−y_j||² = 0`，`(1+0)⁻¹ = 1`，`Σ = n(n−1)`，于是 `q_{ij} = 1/(n(n−1)) = p_{ij}`，`KL(P‖Q) = 0`——代价函数的全局最小。即**坍缩解给出 P=Q，KL=0**。

是否还有别的非平凡解达到同等代价？要使 `KL=0` 需 `q_{ij}=p_{ij}` 对所有 `i≠j` 成立，即所有 n 个点两两等距 `||y_i−y_j||` 相同。在 2 维嵌入空间里，多于 3 个点无法两两等距（2D 中等距点集至多 3 个，即正三角形）；唯一的 n 点两两“等距”构型就是退化情形——全部重合（距离全 0）。故对 `n>3` 的真实数据集，**唯一可达的全局最优就是全部坍缩到一点**。

**动力学层面也支持坍缩。** t-SNE 的梯度为

```
dC/dy_i = 4 Σ_j (p_{ij} − q_{ij})(y_i − y_j)(1 + ||y_i−y_j||²)⁻¹.
```

初始时各 `y_i` 由原点附近的小各向同性高斯采样，间距小，`q_{ij}` 近似均匀 ≈ `p_{ij}`，故初始梯度小；而把所有点拉向原点、缩小所有 `||y_i−y_j||` 会让 `(1+||·||²)⁻¹↑` 且使 `q_{ij}` 趋向 `p_{ij}`，持续降低代价。因此梯度下降会把点群整体吸向原点并互相靠拢，最终所有 n 个点塌成**原点处的一个紧密团点（单一 blob）**，点间距离趋于 0。

最终形态：**低维图退化为一团挤在原点的点云，无任何簇、无任何邻域结构**——这恰好是“输入相似度毫无结构（P 均匀）”的忠实反映：嵌入忠实但没有信息。

---

## 6. 一句话总结

目标困惑度 `> n−1` 越过了 n−1 个支撑点所能承载的最大熵 `log₂(n−1)`，二分搜索因永远够不到目标而把 `σ_i → ∞`，使 `p_{·|i}` 退化为 n−1 点上的均匀分布；对称化后 P 退化为完全图上的均匀相似度（所有点对等相似、无结构）；低维嵌入则把这种“无结构”忠实复刻为**全部点坍缩到原点处一个点的退化解**，`KL(P‖Q)→0`，任何聚类或拓扑信息全部丧失。
