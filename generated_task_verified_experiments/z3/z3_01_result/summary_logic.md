# 位爆破（bit-blasting）对固定位宽 BV 可靠完备、对无界整数不适用 —— 逻辑推导

> 材料依据：Z3 论文（de Moura & Bjørner, 2008）§3「Theory Solvers」明确写道：「The fixed-sized bit-vectors theory applies bit-blasting to all bit-vector operations, but equality.」即 Z3 把固定位宽（fixed-size）位向量的所有运算都位爆破为布尔电路，唯独（逐位）等价不爆破（交给 congruence closure 处理，属优化，不影响正确性）。下面仅由逻辑推理说明此归约为何对 BV 既可靠又完备，而对无界整数行不通。

---

## 一、记号与归约的定义

设有一个 BV 公式 `φ`，其中每个变量都是 **固定位宽 n** 的位向量。位爆破构造一个布尔公式 `φ'`：

1. 把每个 n 位向量变量 `x` 展开为 n 个布尔变量 `x_0, …, x_{n-1}`（即它的二进制位）。
2. 把每个 BV 运算 `f`（如 `+、*、<<、&` 等）编译成一个布尔电路 `C_f`，使其在逐位输入上计算出的输出，恰好等于该 BV 运算按其语义定义的输出位。

记 `B` 为「把位模式解释回 n 位向量」的双射：`B : {0,1}^n ↔ BV_n`。

可靠（sound）= `SAT(φ') ⟹ SAT(φ)`（布尔可满足 ⇒ 原公式可满足）。
完备（complete）= `SAT(φ) ⟹ SAT(φ')`（原公式可满足 ⇒ 布尔可满足）。
二者合起来即 `SAT(φ) ⇔ SAT(φ')`：归约保真。

---

## 二、为什么对固定位宽 BV 既可靠又完备

### 1. 关键前提：BV_n 是一个**有限域**，且运算都是该有限域上的**全函数**

固定位宽 n 的位向量取值集合 `BV_n = {0,…,2^n−1}` 大小恰为 `2^n`，**有限**。
BV 运算的语义是**模 `2^n` 的环绕算术**（modular wrap-around）：例如 `x + y` 的结果是 `(x+y) mod 2^n`，最高位进位被自然丢弃。因此每个 BV 运算 `f : BV_n^k → BV_n` 都是 **`BV_n` 上的一个完全确定的、有界的全函数**——给定输入位，每一位的输出位都是一个确定的布尔函数（可由标准电路给出：半加器/全加器构成行波进位加法器、乘法由组合乘法器电路给出、移位由多路选择器给出，等等）。

> 这一步是整个论证的根基：**正因为域有限、运算有限且全定义，才存在一个有限的布尔电路精确实现它。**

### 2. 电路与运算语义逐位吻合 —— 保真映射

`C_f` 的构造规则保证：对任意位模式输入 `b⃗`，
`bits(f(B(b⃗))) = C_f(b⃗)`。
即电路的输出位串，与「先按 BV 语义算出结果、再取其二进制位」完全一致。这是**按定义成立**的恒等式——电路就是运算的位级定义本身。

### 3. 可靠性（sound）

若 `φ'` 可满足，取其一个满足赋值，把每个向量变量对应的 n 个布尔变量拼回位模式、再用 `B` 解释成 `BV_n` 中的值。由于每个子电路 `C_f` 与对应运算语义逐位吻合，原公式 `φ` 中每个运算/约束的求值结果与 `φ'` 中对应电路的布尔值相同。故 `φ` 亦被满足。**`SAT(φ') ⟹ SAT(φ)`。**

### 4. 完备性（complete）

若 `φ` 可满足，取其一个 `BV_n` 模型，把每个向量变量的取值写成 n 位二进制，得到一组布尔赋值。由逐位吻合性，`φ'` 中每个电路取值与 `φ` 中对应运算一致，故 `φ'` 被满足。**`SAT(φ) ⟹ SAT(φ')`。**

二者合并：`SAT(φ) ⇔ SAT(φ')`，归约保真，因此既可靠又完备。

### 5. 为什么这是「有限、可判定」的 —— 与无界情形的分水岭

- 变量数有限：`n × (向量个数)` 个布尔变量。
- 电路规模有限：每个运算的电路尺寸是 `n` 的多项式（加法线性、乘法 `O(n^2)` 等），公式总规模有限。
- 于是 `φ'` 是一个**有限的 CNF/布尔公式**，可交给 SAT 求解器。SAT 对有限布尔变量是**可判定**的（完备的决策过程，必停），因此整个过程必停、给出确定的 yes/no。

> 一句话：BV 的有限域 + 运算的有限全函数语义 ⟹ 存在有限且保真的布尔编码 ⟹ 归约到有限 SAT 既可靠又完备且可判定。

（注：Z3 对「等价」不爆破、改用 congruence closure，纯属效率优化——逐位 `XNOR ∧ AND` 也能正确爆破等价，省去它只是避免电路膨胀，与可靠性/完备性无关。）

## 三、为什么同样的方法对【无界】整数算术不适用

把同样的位爆破思路套到无界整数 `ℤ` 上，会在三个层面同时崩溃，而根本差异是**域的无界性**。

### 1. 没有「n」可展开 —— 编码本身就无限

无界整数取值于 `ℤ`，没有固定位宽。要把任意整数写成二进制需**无穷多位**（符号位/数值位都无上界）。于是「把每个整数变量展开为 n 个布尔变量」这一步**没有有限的 n 可用**：

- 若强行不设上界 ⟹ 产生**无穷多个布尔变量、无穷大的电路**，不是 SAT 求解器能接受的有限输入，归约根本无法产出可运行的实例。
- 若强行取一个固定上界 k ⟹ 你得到的不再是 `ℤ`，而是 `BV_k`（模 `2^k` 环绕算术）——这是**另一个理论**，语义已被偷换。

### 2. 运算不再是「有限域上的有界全函数」 —— 电路无法有限实现

无界整数算术**不是模运算**：两个整数相乘，结果位数随输入增长而无上界（`a×b` 可任意大），不存在一个固定大小的电路能对**所有**整数输入输出「真正的」乘积而不截断。BV 之所以能爆破，靠的正是「最高位进位丢弃 = mod 2^n」这一把无限增长强制截断为有限的全函数语义；无界整数没有这种截断许可，因此不存在与整数乘法逐位吻合的**有限**电路。

### 3. 可判定性层面：即便编码做出来，也无完备可判定过程

- 含乘法的无界整数算术（`ℤ; +, ×, ≤` 的一阶理论，即真算术）**不可判定**（Gödel 不完备性 / Matiyasevich 对 Hilbert 第十问题的解答：Diophantine 可满足性不可判定）。
- 不可判定意味着**任何**机械过程都不可能既可靠又完备地判定它。归约到有限 SAT 必然二选一地失败：
  - 取有限位宽 ⟹ 截断 ⟹ 语义变成模算术 ⟹ **不可靠**（会承认在 `ℤ` 中本不可满足、但在 `mod 2^k` 下可满足的公式，例如溢出归零的式子）。
  - 不取有限位宽 ⟹ 无限公式 ⟹ SAT 不终止 ⟹ **不完备**（给不出答案）。

（即便只含加法的 Presburger 算术是可判定的，但其判定靠的是 **Omega test / 基于 Simplex 的整数算术求解器**这类专门过程——正是 Z3 论文里与 bit-vectors 并列的「linear arithmetic solver」——而**不是**位爆破。这反过来说明：无界整数即便在可判定的线性片段，也走不了位爆破这条路。）

---

## 四、根本差异一句话总结

| | 固定位宽 BV | 无界整数 |
|---|---|---|
| 取值域 | 有限，`2^n` 个 | 无限，`ℤ` |
| 运算语义 | 模 `2^n` 环绕，**有限域上的有界全函数** | 真·整数算术，结果无界增长 |
| 二进制展开 | **n 位即可**（有限） | 需**无穷多位**（无有限 n） |
| 电路实现 | 每运算有**有限**电路逐位吻合 | 无有限电路能精确实现乘法等 |
| 归约产物 | **有限**布尔公式 ⟹ 可交 SAT | 无限公式 / 截断后变模算术 |
| 可判定性 | 归约到有限 SAT，**可判定** | 含乘法则**不可判定**；线性片段可判定但**不经位爆破** |

**根本差异**：位爆破的正确性与有限性，完全仰赖于「固定位宽 ⟹ 有限域 ⟹ 运算为有限域上有界全函数 ⟹ 存在有限且逐位保真的布尔电路 ⟹ 归约到有限 SAT（一个完备的可判定过程）」这条因果链。无界整数在链条的第一节就断裂——域无限、无有限 n、运算无界增长——使保真编码要么无限（不完备、不可运行），要么靠截断沦为模算术（不可靠）；且含乘法的整数算术本就不可判定，注定不存在可靠又完备的有限归约。这正是 BV 可爆破而 `ℤ` 不可爆破的根因。
