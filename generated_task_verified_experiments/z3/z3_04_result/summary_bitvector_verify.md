# 位向量（BitVec）定宽算术性质验证 — 结论

## 任务与方法

用 z3-solver (v4.8.12) 的位向量（BitVec）理论，对固定位宽的无符号定宽算术性质做自动验证。材料为 `z3_material/z3_demoura_bjorner_2008.pdf`（de Moura & Bjørner, *Z3: An Efficient SMT Solver*, 2008），其中描述了 z3 对 bit-vector 理论通过 bit-blasting 归约到 SAT 的求解策略——本实验观察到的求解时间正反映了这一特征。

固定设置：
- 位宽 `w = 8`（主），并附 `w = 16` 以验证结论与位宽无关。
- 性质集合：加法无符号溢出存在性 (a)、加减可消去性 (b)、乘除可消去性反例 (c1)、无溢出前提下的乘除可消去性 (c2)。
- 求解器：默认 `Solver()`（CPU-only，无 GPU，无超参自变量）。
- 验证范式：
  - **证明成立**：断言性质的否定 → 判定 **UNSAT**（无反例即证明）。
  - **找反例**：直接求满足 → 判定 **SAT** 并取模型。

脚本：`bv_verify.py`，原始计时结果：`results.json`。

## 性质编码

- **(a)** 是否存在 `x, y` 使 `x + y`（无符号）溢出？
  编码：`s == x + y` 且 `s <_u x`（无符号下结果小于操作数 ⇔ 最高位有进位 ⇔ 溢出）。求满足解。
- **(b)** `(x + y) − y == x` 是否对全部 `x, y` 成立？
  断言否定 `(x + y) − y != x`，求满足解（期望 UNSAT）。
- **(c1)** 乘法 wraparound 反例：`(x * y) / y == x`（无符号除，`y != 0`）是否总成立？
  断言否定 `UDiv(x*y, y) != x` 并 `y != 0`，求满足解（期望 SAT，给出 wraparound 反例）。
- **(c2)** 乘法 wraparound 条件（正向性质）：若 `x*y` 不发生无符号溢出且 `y != 0`，则 `(x*y)/y == x`。
  无符号乘溢出判据：`overflow ⇔ (x != 0) ∧ (UDiv(x*y, x) != y)`（用模 `2^w` 乘积判定）。
  断言：`y != 0 ∧ ¬overflow ∧ UDiv(x*y, y) != x`，求满足解（期望 UNSAT，即证明）。

## 结果汇总

| 位宽 | 性质 | sat/unsat | 判定 | 求解时间 | 反例/见证 |
|---|---|---|---|---|---|
| 8  | (a)  溢出存在性      | **SAT**   | 找到见证 | 42.6 ms | x=1, y=255 → 和=256 (mod 256 = 0) |
| 8  | (b)  (x+y)−y==x 全体 | **UNSAT** | 证明成立 | 2.4 ms  | — |
| 8  | (c1) (x*y)/y==x 反例 | **SAT**   | 找到反例 | 17.4 ms | x=202, y=9 → (202·9=1818 mod 256=26), 26/9=2 ≠ 202 |
| 8  | (c2) 无溢出⇒可消去   | **UNSAT** | 证明成立 | 58.1 ms | — |
| 16 | (a)  溢出存在性      | **SAT**   | 找到见证 | 9.8 ms  | x=1, y=65535 → 和=65536 (mod 65536 = 0) |
| 16 | (b)  (x+y)−y==x 全体 | **UNSAT** | 证明成立 | 3.8 ms  | — |
| 16 | (c1) (x*y)/y==x 反例 | **SAT**   | 找到反例 | 37.3 ms | x=65199, y=33 → (65199·33 mod 65536=54415), 54415/33=1648 ≠ 65199 |
| 16 | (c2) 无溢出⇒可消去   | **UNSAT** | 证明成立 | 60890.9 ms | — |

## 结论

1. **(a) 加法无符号溢出存在性 — SAT（找到见证）。** 8 位与 16 位都存在溢出实例。8 位最小见证 `x=1, y=255`：真值和 256 超出 `[0, 255]`，模 256 后回卷为 0（`0 <_u 1` 即溢出标志成立）。这证实定宽无符号加法确有溢出现象，且 z3 能直接给出见证。

2. **(b) 加减可消去性 — 证明成立（UNSAT）。** `(x + y) − y == x` 在模 `2^w` 算术下对全体 `x, y` 成立。原因是 BV 加/减法均为模 `2^w` 运算，模运算构成 Abel 群，每个元素可加可逆，故 `+y` 与 `−y` 互为逆运算、可消去。否定被判定 UNSAT，无反例。两个位宽下均 < 4 ms，bit-blasting 后本质是一组等价传播，极快。

3. **(c1) 乘除不可无条件消去 — 找到反例（SAT）。** 与 (b) 不同，`(x * y) / y == x` **并非**恒成立。8 位反例 `x=202, y=9`：真值积 `202·9 = 1818`，模 256 回卷为 `26`，再无符号除以 9 得 `2`，与原值 `202` 不等。根因是乘法 wraparound 丢失了高位信息，而除法无法恢复——`/y` 不是 `*y` 的逆（除非乘积不溢出）。16 位反例同理（积回卷为 54415，除以 33 得 1648 ≠ 65199）。

4. **(c2) 无溢出前提下的乘除可消去性 — 证明成立（UNSAT）。** 加上「`x*y` 不发生无符号溢出」的前提后，`(x*y)/y == x`（`y != 0`）成立。这是 (c1) 的精确「分界」：wraparound 正是使乘除消去性失效的唯一原因；一旦排除 wraparound，模乘积即等于真值积，整数除法可恢复原值。z3 通过否定 UNSAT 证实，与数学预期一致。

5. **求解时间与位宽的关系。** (b) 在两种位宽下都极快（< 4 ms），因为纯加减的 bit-blast 网络规模随位宽线性且结构简单。(c2) 从 8 位的 58 ms 跃升到 16 位的 ~60.9 s，跨越 3 个数量级——这正体现 de Moura–Bjørner 论文中所述 bit-blasting 对**含乘法**的电路呈近指数膨胀：乘法电路规模 O(w²)，16 位下 SAT 求解器需处理数万变量/子句的爆破网络。(a)、(c1) 因只需求一个见证（非完备证明），求解迅速。

## 验证结果一览（证明成立 / 找到反例）

| 性质 | 类型 | 结果 |
|---|---|---|
| (a) 存在 x,y 使 x+y 无符号溢出 | 找反例 | **找到见证**（SAT）：8 位 x=1,y=255；16 位 x=1,y=65535 |
| (b) (x+y)−y == x 对全体 x,y | 证明成立 | **证明成立**（UNSAT） |
| (c1) (x*y)/y == x 对全体 x,y(y≠0) | 找反例 | **找到反例**（SAT）：8 位 x=202,y=9；16 位 x=65199,y=33 |

总运行时间约 61 秒（其中 16 位 (c2) 占 ~61 s），远低于 30 分钟上限；全程 CPU-only。
