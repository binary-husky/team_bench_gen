# SimHash + 汉明阈值 近重复检测：precision / recall 随阈值 T 的变化

## 1. 任务与理论依据

参考材料：Charikar 2002, *Similarity Estimation Techniques from Rounding Algorithms*（STOC'02）。
其核心结论：对向量用「随机超平面投影取符号」构造的 b 位 SimHash 草图，每一位相等的概率为

> Pr[某位相等] = 1 − θ/π ，其中 θ = arccos(cosine similarity)。

因此两位 b 位草图的**期望汉明距离**

> E[Hamming(x,y)] = b · θ / π = b · arccos(cos) / π 。

推论：
- 真近重复对（余弦 ~0.95+，θ 小）→ 汉明距离小；
- 无关对（余弦 ~0，θ ≈ π/2）→ 汉明距离 ≈ b/2 = 128。

判定规则：**Hamming(x,y) ≤ T ⇒ 预测为近重复**。T 是本实验唯一自变量。

## 2. 固定设置（除 T 外全部固定）

| 参数 | 值 | 说明 |
|---|---|---|
| d | 100 | 向量维 |
| b | 256 | SimHash 位数 |
| 种子 | 42 | `np.random.default_rng(42)`，全程固定 |
| N_base | 300 | 独立基础向量，~N(0,I)，单位化 |
| N_near | 150 | 对前 150 个基础向量各构造 1 个近重复伴 |
| 语料规模 N | 450 | 300 基础 + 150 近重复伴 |
| 配对总数 | C(450,2)=101 025 | 全配对，无遗漏 |
| 近重复构造 | v′ = a·v + √(1−a²)·z | z 为与 v 正交的单位向量 ⇒ cos(v,v′)=a 精确 |
| a 采样范围 | U(0.90, 0.99) | 使近重复余弦在 0.90–0.99 间分散 |
| 真值阈值 | cos ≥ 0.85 | 由实测余弦判定（构造对全部满足） |

SimHash：256 个独立标准正态投影向量 P∈R^{d×b}，位 = (X·P > 0)。配对汉明距离用 packed-bit XOR+popcount 计算。

## 3. 语料与汉明距离的实证分布

| 配对类别 | 数量 | 余弦 min/mean/max | 汉明 min/mean/max |
|---|---|---|---|
| 真近重复（cos≥0.85） | 150 | 0.901 / 0.948 / 0.989 | 11 / 25.6 / 46 |
| 其他（无关）对 | 100 875 | −0.419 / −0.001 / 0.390 | 79 / 128.1 / 180 |

理论值对照：E[Ham]@cos=0.90→36.8，@cos=0.99→11.5，@cos=0→128.0。实测均值 25.6 与 128.1 与理论高度吻合。
**两类汉明距离完全分离**：真近重复最大 46，无关对最小 79，间隔 [46, 79] 内无任何配对 → 存在“零误差”阈值带。

## 5. 曲线特征（三个区段）

1. **召回爬升区（T ≈ 11–46）**：precision ≡ 1.000（无假阳性），recall 从 0.013 单调升至 1.000。
   - recall=0.5 落在 T≈25；recall=1.0 首次达成于 **T=46**（恰好等于真近重复的汉明最大值）。
   - 此区间内随 T 增大只是把更多（汉明更小的）真近重复对纳入，FP 始终为 0。
2. **完美平台区（46 ≤ T ≤ 78）**：precision=1.000、recall=1.000、F1=1.000。
   - 因为真近重复汉明 ≤46，无关对汉明 ≥79，二者之间留有 [46,79] 的安全间隔，任何落在此带的 T 都给出零误差分类。
3. **精度崩塌区（T ≥ 79）**：recall 恒为 1.000，但无关对（汉明均值 128、标准差 ≈8 的中心分布）开始落入 ≤T，FP 激增，precision 急剧下降。
   - precision=0.987@T=79 → 0.721@T=90 → 0.158@T=100 → 0.003@T=128。
   - T=128（=b/2，等价于“随机猜”）时 precision 退化为先验 150/101025≈0.0015，检测器完全失效。

**最佳工作点**：T∈[46,78] 给出 precision=recall=F1=1.000。Best-F1（按 (F1, recall) 选）落在 **T=46**。

## 6. 结论

- 在本设置（d=100, b=256, 近重复 cos≈0.90–0.99, 无关对 cos≈0）下，SimHash 把余弦相似度干净地映射成汉明距离：真近重复汉明 11–46，无关对汉明 79–180，**两类完全分离**。
- precision 随 T 单调不增、recall 随 T 单调不减，符合阈值分类器的一般规律；但因两类分布分离良好，存在一个**宽达 33（T∈[46,78]）的零误差平台**。
- T 太小（<46）只损失 recall（漏检高相似度不够的近重复），precision 不受影响；T 太大（≥79）只损失 precision（把无关对错判为近重复），recall 已饱和。换言之：**欠阈值→漏检，过阈值→误报**，最优点在中间带。
- 经验法则 T ≈ b·arccos(cos)/π 与实测吻合：要召回 cos≥0.95（θ≈0.3176, E[Ham]≈25.9）的近重复，T 应设在 ~26 以上；要避开 cos≈0 的无关对（E[Ham]≈128），T 应远小于 128。本实验最优带 T≈46–78 正落在这两界之间。
