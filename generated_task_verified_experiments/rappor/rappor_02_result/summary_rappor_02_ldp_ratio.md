# RAPPOR PRR 每比特 LDP 概率比上界 —— 实证验证

## 1. 实验设置

按 `task.md` 要求，**进程内**从零实现 RAPPOR 的 Permanent Randomized Response (PRR)：

- 对一个 Bloom 比特 `b`，以概率 `f` 替换为均匀随机比特（`Ber(0.5)`），否则保留 `b`。
- 等价边际概率：
  - `Pr[PRR=1 | b=1] = 1 − f/2`
  - `Pr[PRR=1 | b=0] = f/2`
- 理论每比特局部差分隐私上界：
  - `r = (2−f)/f = e^{ε_perm}`，即 `ε_perm = ln((2−f)/f)`。

**固定参数**（未更改）：

- 两个相邻输入群体：`b=1` 组与 `b=0` 组，每组 `N = 2×10^5` 个比特样本（≥ 1e5）。
- 参数网格：`f ∈ {0.10, 0.25, 0.50, 0.75, 0.90}`。
- 每个 `f` 用 **7 个不同随机种子**（0–6，≥ 5）重复，对各组求经验概率的均值。
- 经验比值 `r̂ = mean_seeds(Pr̂[PRR=1|b=1]) / mean_seeds(Pr̂[PRR=1|b=0])`，经验 `ε̂ = ln(r̂)`。

实现见 `prr_experiment.py`（NumPy 向量化，`numpy 1.26.4`）。

### 采样噪声检验（log-ratio 的 delta-method z 值）

用 `SE(ln r̂) = sqrt((1−p1)/(p1·N) + (1−p0)/(p0·N))`，对 `ln(r̂) − ln((2−f)/f)` 求 z：

| f    | z(log-ratio) |
|------|--------------|
| 0.10 | 1.13         |
| 0.25 | 0.33         |
| 0.50 | 0.13         |
| 0.75 | 0.19         |
| 0.90 | 0.25         |

所有 `|z| < 2`，经验比值与理论值 `(2−f)/f` 的偏差完全落在采样噪声内。各 `f` 下经验概率 `p̂1` 在 7 个种子间的标准差也与二项 SE `sqrt(p(1−p)/N)` 一致（见 `prr_experiment.py` 输出），佐证估计无系统偏差。

## 3. 结论要点

1. **经验比值等于理论上界**：对网格中每个 `f`，`r̂ = Pr̂[1|b=1]/Pr̂[1|b=0]` 与 `(2−f)/f` 的最大相对误差仅 1.11%（出现在 `f=0.10`，那里 `p0≈0.05` 较小故比值方差最大），且 log-ratio 的 z 值全部在 ±2 内 —— 偏差在采样噪声内，**无法拒绝** `r̂ = (2−f)/f`。对应地 `ε̂ = ln(r̂)` 与 `ε_perm = ln((2−f)/f)` 一致（最大差 ≈ 0.011 nat）。

2. **`f` 的单调性 / 隐私-效用权衡验证**：
   - `f↑` ⇒ `(2−f)/f↓`：从 `f=0.10` 的 19.0 单调降到 `f=0.90` 的 1.222。
   - ⇒ `ε_perm↓`：从 2.94 nat 降到 0.20 nat。
   - ⇒ **隐私越强**（`f` 越大，PRR 越频繁地把比特替换成纯随机比特，输出对原 `b` 的依赖越弱；极端 `f→1` 时 `Pr[1|b=1]=Pr[1|b=0]=0.5`，比值→1、ε→0，即完全无信息泄露）。
   - 反之 `f↓` ⇒ ε↑ ⇒ **隐私越弱**（`f→0` 时几乎不扰动，比值→∞、ε→∞，无隐私）。

3. **LDP 界的成立**：PRR 机制满足 `Pr[PRR=1|b=1] / Pr[PRR=1|b=0] = (1−f/2)/(f/2) = (2−f)/f = e^{ε_perm}`，且同理对称地 `Pr[PRR=0|b=0]/Pr[PRR=0|b=1] = (2−f)/f`。因此对任意相邻输入（`b=0↔b=1`）与任意输出，似然比均被 `e^{ε_perm}` 界住，PRR 提供每比特 `ε_perm = ln((2−f)/f)`-局部差分隐私。实证与该上界在采样噪声内吻合，**验证了 PRR 的每比特 LDP 界**。

> 注：这是 PRR 单比特的界；完整 RAPPOR 还叠加瞬时随机响应 (IRR) 与队列/cohort 机制，整体隐私由 `ε_perm`、`ε_IRR` 等组合刻画，本实验只验证 PRR 这一层的每比特上界。
