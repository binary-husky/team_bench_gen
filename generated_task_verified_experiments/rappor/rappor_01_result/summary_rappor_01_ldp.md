# RAPPOR PRR 的局部差分隐私与 ε 推导

> 任务：仅逻辑推理——PRR 对每比特以概率 f 替换为随机比特、1−f 保留；推导 PRR 给局部 DP 的 ε；IRR 的 ε；f/ε 的隐私-效用关系。
> 依据 Erlingsson-Pihur-Korolova, *RAPPOR* (CCS 2014)。

## (a) 局部 DP 定义

对相邻输入（同一比特位 `b=0` vs `b=1`），任意输出 `o` 满足 `Pr[out=o | b=1] / Pr[out=o | b=0] ≤ e^ε`（双向，取倒数亦然）。

## (b) PRR 的 ε

PRR 概率：`Pr[PRR=1|b=1]=1−f/2`，`Pr[PRR=1|b=0]=f/2`。
- 输出 `o=1` 的概率比 = `(1−f/2)/(f/2) = (2−f)/f`。
- 输出 `o=0` 的比 = 其倒数 `(f/2)/(1−f/2) = f/(2−f)`。
- 两输出均被 `(2−f)/f` 控制 ⟹ **`ε_perm = ln((2−f)/f)`**。
  - `f=0.5` ⟹ `(2−0.5)/0.5 = 3` ⟹ `ε_perm = ln 3 ≈ 1.10`。

## (c) IRR 的 ε

IRR：`Pr[IRR=1|PRR=1]=q`，`Pr[IRR=1|PRR=0]=p`。同理输出 `1` 的比 = `q/p`，输出 `0` 的比 = `(1−q)/(1−p)`；IRR 的 ε = `max(ln(q/p), ln((1−q)/(1−p)))`。端到端 ε（PRR∘IRR）按串行组合 ≈ `ε_perm + ε_irr`。

## (d) f/ε 是隐私-效用旋钮

`f` 大 ⟹ `ε_perm` 小（更私密）但 PRR 与原比特相关度低、聚合估计方差大（效用差）；`f` 小 ⟹ `ε` 大、效用好但隐私弱。故 `f`（及 `p,q`）是隐私-效用旋钮。

## 一句话总结

PRR 每比特概率比 = `(2−f)/f` ⟹ `ε_perm=ln((2−f)/f)`（f=0.5⇒ln3≈1.10）；IRR 的 `ε=max(ln(q/p),ln((1−q)/(1−p)))`；f/ε 是隐私-效用旋钮——f 大更私密但效用差。
