读给定材料，做实验，写结论。

用 Python + **NetworkX**（精确 `betweenness_centrality`）。NetworkX 可直接生成多种结构图：路径图 `nx.path_graph`、环图 `nx.cycle_graph`、星图 `nx.star_graph`、二维网格 `nx.grid_2d_graph`，以及带社区结构的图（如 `nx.random_partition_graph` 或 SBM `nx.stochastic_block_model`）。

研究目标：**对比不同图拓扑下介数中心性的分布形态，验证介数对图结构高度敏感。**

固定实验设置（不要更改）：
- 取节点数相近（如 **n ≈ 50–200**）的多种拓扑：路径图 `P_n`、环图 `C_n`、星图 `S_n`、二维网格、随机分区图（带社区）各一张。
- 对每张图算精确介数（`normalized=False` 或统一约定）。
- 记录每张图的介数分布统计：**最大介数** `max BC`、**最大介数节点位置**（如路径图应在中间、星图在中心）、**基尼系数（Gini）**衡量分布集中度。

把以下内容写到 `./summary_betweenness_05_structure.md`：
1. 一张表：每种拓扑的 `max BC`、最大介数节点、基尼系数；并附每种的介数分布直方图（或描述）。
2. 结论要点：
   - **路径图**：介数从两端向中间线性递增，中间节点最高、两端为 0；
   - **环图**：所有节点介数相等且较低（对称）；
   - **星图**：中心节点介数极高（占据几乎所有最短路）、其余节点为 0，基尼系数接近 1；
   - **网格**：介数分布较平缓、中心区域略高；
   - **社区图**：社区"桥"节点（连接不同社区的节点）介数显著高于社区内部节点。
   整轮 **< 30 分钟**。
