# 社区数量与社区规模分布随图规模的变化

## 任务与设置

考察 LFR 基准图上 Louvain 检测的社区数、模块度 Q、规模分布与集中度如何随图规模 n 变化。

固定参数：`tau1=3, tau2=1.5, mu=0.4, average_degree=6, min_community=20, seed=0`。唯一自变量为 n ∈ {500, 1000, 2000, 4000}。

方法：`networkx.LFR_benchmark_graph(...)` 生成图；`community.best_partition(G, random_state=0)`（python-louvain 0.16）检测社区；模块度用同一库的 `modularity`。集中度指标为基尼系数（按升序排序的标准公式 G = (2·Σi·xᵢ − (N+1)·Σx)/(N·Σx)）。规模分布按降序记录。环境：CPU-only，networkx 2.8.8，单种子 seed=0（遵照任务固定设置）。

## 结论：随 n 的变化趋势

**1. 社区数 k 随 n 增长，但亚线性且非严格单调。**
k 从 15 → 20 → 22 → 35。整体随 n 上升，但 500→1000→2000 段增长很平缓（15→20→22），到 4000 才明显跳到 35。即社区数增长慢于图规模：平均社区规模 n/k 从 33 → 50 → 91 → 114，社区随图变大而"膨胀"，而非等比例地增多。这符合 Louvain 的层次性特点——更倾向于把节点并入已有大社区来提升局部模块度增益，而非无限细分。

**2. 模块度 Q 上升后趋于饱和（~0.5–0.56）。**
Q: 0.45 → 0.56 → 0.51 → 0.53。n=500 时最低（小图上 planted 结构信号偏弱、被噪声边稀释，Q≈0.45）；n≥1000 后稳定在 0.5–0.56 区间并基本饱和。注意 n=1000 处 Q 最高（0.56）而 n=2000 反而略降，说明单种子下存在随机波动，但量级稳定。结论：图足够大时模块度不再随 n 显著抬升。

**3. 规模分布的离散度与不均衡度随 n 上升。**
基尼系数从 0.23 → 0.26 → 0.35 → 0.38 单调上升；CV 从 0.43 → 0.47 → 0.62 → 0.70 同步上升。n 小时分布相对均匀（n=500 几乎所有社区在 15–65 之间，仅 1 个孤立点）；n 大时出现明显的"长尾"——少数超大社区（n=4000 时最大 318、占全图 ~8%）与一批小社区/单点（5 个规模=1）并存。最大社区规模近似随 n 线性增长（65 → 95 → 200 → 318），而最小社区恒为 1（Louvain 倾向甩出孤立弱连节点）。

**4. 层级展开（hierarchical unfolding）的体现。**
Blondel 等 (2008) 的层次性指 Louvain 在 phase 1 把叶子节点并入局部社区、phase 2 把社区作为超节点继续聚合，从而在不同尺度上产生社区。本实验的"n 增大"虽是改变图规模而非在同一图上展开层次，但观察到的现象与之呼应：图越大，Louvain 越能形成跨多个尺度的社区结构——顶端是少数大社区（深层聚合的产物），尾端是未充分合并的小社区/单点（浅层未消化的碎片）。因此规模分布随 n 变得越来越"重尾/分层"，集中度（基尼、CV）随之上升。

**总括**：在固定 LFR 参数与 seed=0 下，随 n 从 500 增至 4000——(i) 检测社区数亚线性增长（平均规模随之增大）；(ii) 模块度快速饱和于 0.5–0.56；(iii) 规模分布由相对均匀走向重尾不均衡，基尼系数 0.23→0.38、CV 0.43→0.70，最大社区线性增大而最小恒为 1。这组变化与 Louvain 的层次式聚合机制一致：图规模越大，社区在多尺度上分化越显著。
