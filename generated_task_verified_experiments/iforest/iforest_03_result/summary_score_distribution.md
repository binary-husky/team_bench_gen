# Isolation Forest 异常分（及路径长度）分布：正常点 vs 异常点

## 1. 实验设置（固定，无自变量）

按论文 Liu et al. 2008《Isolation Forest》及其异常分定义，在固定设置下做一次分布测量。

**数据生成（固定种子 `RandomState(0)`）：**
- 正常簇：2000 个点，5 维标准高斯 `N(0, I)`（单一密集簇）。
- 注入离群：`round(2000*0.02)=40` 个点，5 维，在 `[6, 10]^5` 均匀采样（远离正常簇，约 1.96%）。
- 合计 2040 点；真实标签 `y=0`(正常) / `y=1`(异常)。
- 维度 d=5，离群在每一维都偏离 ≥6σ，属于明确可分离的“远端”离群。

**模型（固定）：** `IsolationForest(n_estimators=100, max_samples=ψ=256, contamination='auto', random_state=0)`，CPU 拟合。

**异常分定义（论文）：**
- `s(x) = 2^{-E[h(x)] / c(ψ)}`，`E[h(x)]` 为该点在所有 iT 树上的平均路径长度。
- `c(n) = 2·H(n-1) - 2(n-1)/n`，`H(i)≈ln(i)+0.5772156649`。本实验 `c(256)=10.2448`。
- `s→1` 表示异常（路径短、易隔离）；`s≈0.5` 及以下表示正常。

> 实现备注：sklearn 的 `score_samples` 返回的正是论文 `s` 的相反数（即 `score_samples = -s`），故 `s = -score_samples`；`decision_function = score_samples - offset_`，本例 `offset_=-0.5`，即 `decision_function = 0.5 - s`（越负越异常）。两种口径下分布形状完全一致，仅相差一个线性变换与符号。下面主报告论文 `s`，并附 `decision_function` 口径。

## 2. 异常分分布对比（论文口径 `s = 2^{-E[h]/c}`，越大越异常）

| 统计量 | 正常点 (n=2000) | 异常点 (n=40) |
|---|---|---|
| 均值 | 0.4197 | 0.7750 |
| 中位数 | 0.4114 | 0.7746 |
| 标准差 | 0.0410 | 0.0104 |
| min | 0.3542 | 0.7497 |
| Q05 | 0.3682 | 0.7577 |
| Q25 | 0.3883 | 0.7681 |
| Q75 | 0.4439 | 0.7825 |
| Q95 | 0.4988 | 0.7897 |
| max | 0.6188 | 0.7902 |

**关键观察：**
- 两组分布**几乎完全不重叠**。正常点 s 全域 `[0.354, 0.619]`，异常点 s 全域 `[0.750, 0.790]`；正常点 95 分位 0.499 仍远低于异常点最小值 0.750，二者之间存在约 `0.13` 的“空带”。
- 异常点 s 均值 0.775 ≈ 论文中“离群明显”区间（s>0.6 且随隔离度趋近 1），正常点均值 0.420 < 0.5，符合论文“s≤0.5 时样本多为正常”的论断。
- 异常点分布极窄（std=0.010，约为正常点 std 的 1/4）：注入的离群点距簇同等远、被同等快地隔离，路径长度几乎一致；正常点因在簇内位置不同（中心 vs 边缘），路径长度方差更大。

## 3. 路径长度 `E[h]` 分布对比

由 `s = 2^{-E[h]/c}` 反解 `E[h] = -c·log2(s)`：

| 统计量 | 正常点 | 异常点 |
|---|---|---|
| `E[h]` 均值 | 12.90 | 3.77 |
| `E[h]` 中位数 | 13.13 | 3.78 |

- 异常点平均路径长度 ≈3.77，**远短于** `c(256)=10.24`，对应 `s>0.5`——只需极少几次随机划分即可被隔离，正是“异常易隔离”的体现。
- 正常点平均路径长度 ≈12.9，**略长于** `c(256)`（`E[h]>c` → `s<0.5`）。这是正常的：`c(n)` 是“不成功搜索”的平均长度，落在密集簇内的点往往走到树深处的叶节点才停止，因而路径可超过 `c(n)`，导致 `s` 落到 0.5 以下。
- 路径长度与异常分的结论一致：异常点隔离快、路径短、s 高；正常点隔离慢、路径长、s 低。

## 4. 可分性：AUC

以真实标签 `y` 计算 ROC-AUC（得分越大越异常）：

| 口径 | AUC |
|---|---|
| 论文 `s = 2^{-E[h]/c}` | **1.0000** |
| `-decision_function`（sklearn 异常分，越大越异常） | **1.0000** |

AUC=1.0：在本固定设置下，正常点与异常点的异常分完全可分，不存在错分。这与第 2 节“分布不重叠、间有空带”一致——任何介于正常点最大值（0.619）与异常点最小值（0.750）之间的阈值都能 100% 正确划分，理论最优阈值约在 `s≈0.68`。

## 5. 结论

1. **分布差异显著且方向符合论文预期**：异常点异常分 `s`（均值 0.775）明显高于正常点（均值 0.420），二者之间有约 0.13 的无重叠空带；正常点 `s<0.5`、异常点 `s>0.5`，与论文“s→1 异常、s≈0.5 正常”一致。
2. **路径长度是分异的根因**：异常点平均路径 `E[h]≈3.8`（短于 `c(256)=10.24`），正常点 `E[h]≈12.9`（长于 `c`），隔离速度的对比直接造就了异常分的分离。
3. **可分性近乎完美**：AUC=1.00，说明在该数据/参数下 IsolationForest 对“远端注入型离群”完全可识别；分离能力同时由分布空带（不仅是均值差异）保证。
4. **稳健性提示**：异常点分数方差极小（std≈0.010），源于离群点同质性；若离群在分布与距离上更分散（贴近簇或部分维度重叠），空带会缩窄、AUC 将下降——本例的完美可分部分得益于离群距簇 ≥6σ 的强设定。
