研究 **3 目标 DTLZ1–DTLZ7 不同 Pareto 前沿几何**对同一份 NSGA-II 的挑战差异。固定算法、固定 budget、固定 M=3，让"前沿几何"成为唯一变量，量化每种几何给算法造成的**收敛**与**多样性**两类损失。

固定实验设置：

- 算法：标准 NSGA-II（二元锦标赛 + 精英保留 + 拥挤距离）
- 种群大小：N = 100
- 交叉：SBX（η_c = 15, p_c = 1.0，遵循 DTLZ 论文）
- 变异：多项式变异（η_m = 20, p_m = 1/n）
- 决策变量：k = 10（即 n = M + k − 1 = 12）
- 独立运行次数：每个问题 31 次
- 终止条件：500 代（统一加长，让 DTLZ3/DTLZ6 这种难问题有机会展示完整动力学）

7 个测试问题（每个对应一种前沿几何，论文 §6.7.1–6.7.7）：

| 问题 | 前沿几何 | 关键特征 | 论文图 |
|---|---|---|---|
| DTLZ1 | 线性超平面 f_1+f_2+f_3=0.5 | 多模态 g（11^k−1 局部前沿） | 6.12 |
| DTLZ2 | 单位球面第一象限（凹） | 单模态 g，球面几何 | 6.14 |
| DTLZ3 | 单位球面第一象限 | g 极度多模态（3^k−1 局部前沿） | 6.16 |
| DTLZ4 | 单位球面第一象限（biased） | α=100 power mapping，密度偏置 | 6.18 |
| DTLZ5 | 退化到 1D 曲线 | θ_i 退化映射（式 6.10） | 6.20 |
| DTLZ6 | 退化到 1D 曲线（deceptive） | g = Σ x_i^0.1，deceptive | 6.22 |
| DTLZ7 | 2^(M−1)=4 段不连通 | sin(3π f_i) 离散化 | 6.24 |

需要计算的指标：

1. **收敛指标**：IGD（每个问题用对应的真实 Pareto 前沿采点 1000 个作为参考集）
2. **多样性指标**：Spacing（解间距离的方差，越小越均匀）、Spread Δ（边界覆盖度）
3. **DTLZ7 专项**：4 段不连通前沿每段内的解数量（理想每段 ~25 个）；31 次 run 中"4 段全有解"的成功率
4. **DTLZ5/6 专项**：最终种群在 (f_1, f_3) 投影上的分布——理想是一条 1D 曲线，统计种群到该曲线的 RMS 距离
5. **DTLZ4 专项**：31 次 run 中落入 "f_3-f_1 plane" / "f_1-f_2 plane" / "entire surface" 三种结果的分布（论文 §6.7.4 明确说三种 outcome 都可能出现）

把 7 问题 × 4 指标的总表 + 各专项诊断写到 `./summary_dtlz_03_front_geometry.md`，并明确回答：**哪一种前沿几何对 NSGA-II 的"收敛"造成最大损失？哪一种对"多样性"造成最大损失？**
