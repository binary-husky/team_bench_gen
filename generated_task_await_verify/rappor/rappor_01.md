背景：RAPPOR 的 **Permanent Randomized Response (PRR)** 对 Bloom 编码向量 `B` 的**每个比特**独立做：以概率 `f` 把该比特替换成一个全新随机比特（0/1 等概率），以概率 `1−f` 保留原值。等价地：

```
Pr[PRR_i = 1 | B_i = 1] = (1−f)·1 + f·(1/2) = 1 − f/2
Pr[PRR_i = 1 | B_i = 0] = (1−f)·0 + f·(1/2) = f/2
```

瞬时随机响应 **IRR**（参数 `p, q`）则满足 `Pr[IRR_i=1 | PRR_i=1] = q`、`Pr[IRR_i=1 | PRR_i=0] = p`。

---

**问题**（唯一一题）：

请从**局部差分隐私（Local DP）的定义**出发，逻辑推理回答：**为什么 PRR（以及 IRR）步骤提供了 ε-局部差分隐私？ε 分别是多少？** 你的回答需覆盖以下推理链（给定材料给出了 RAPPOR 的参数与隐私结论，但需要你自己从 LDP 定义把"ε 是多少、为何满足"推导出来）：

1. **局部 DP 的定义**：对任意两个"相邻"输入（在比特层面即 `B_i = 0` 与 `B_i = 1`），机制输出任意结果 `o` 的概率比 `Pr[out=o | b=1] / Pr[out=o | b=0]` 必须 **≤ e^ε**（对 `o ∈ {0,1}` 都成立）。
2. **PRR 的 ε 推导**：对 `o = 1`，比值 = `(1 − f/2) / (f/2) = (2 − f)/f`；对 `o = 0`，比值 = `(f/2) / (1 − f/2)`，是其倒数。两者都被 `(2 − f)/f` 上界控制。故 **PRR 每比特满足 ε_perm-局部 DP，ε_perm = ln((2 − f)/f)**。代入默认 `f = 0.5` 得 `(2−0.5)/0.5 = 3`，即 **ε_perm = ln 3 ≈ 1.10**。
3. **IRR 的 ε**：类似地，IRR 对 `o=1` 的比值为 `q/p`，故 **IRR 每比特满足 ε_irr-局部 DP，ε_irr = ln(q/p)**。代入默认 `p=0.5, q=0.75` 得 `q/p = 1.5`，即 **ε_irr = ln 1.5 ≈ 0.405**。

最后给出一句结论：**随机响应层提供局部 DP，ε 由输出概率比的对数决定（PRR：ln((2−f)/f)；IRR：ln(q/p)）；`f` 越大（或 `q−p` 间距越大）→ ε 越大 → 隐私越弱。反之 `f` 越小 → ε 越小 → 隐私越强但效用越差。**
