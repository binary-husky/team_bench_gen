    - 选择步骤 (5) 中 `f(u_i,g) = f(x_i,g)`，按照 `f(u_i,g) < f(x_i,g)` 的严格小于判定，**不会发生成功更新**；
    - 因此 `x_i,g` 不会进入 archive A，`F_i` 也不会进入 SF；
    - 当大量个体同时撞上 `F_i = 0`（例如 Cauchy 分布有重尾，概率不低）时，A 不增长、SF 为空或几乎为空、μ_F 在公式 (11) 中 `mean_L(SF)` 无意义或保持不变——**整个种群的探索停滞，参数自适应失效**。
2. **CR 的对比（必须出现）**：当 `CR_i = 0` 时，公式 (4) 中除 `j = j_rand` 外全部继承父代，因此 trial vector 与父代仅在 `j_rand` 一个分量上不同——**trial 与 parent 仍然有差别**，仍可能发生成功更新、仍会进入 SF/SCR、archive 仍会增长。所以对 CR 截断到 [0,1] 不会触发 F 那种"整体冻结"的级联失败。
3. **结论（必须出现）**：JADE 对 `F_i ≤ 0` 必须 regenerate，因为 F 是变异向量的**乘性标量**（其符号决定差分向量的方向，其零值导致整个差分项消失），而 CR 是分量级的**概率**（其符号在 [0,1] 之外无物理意义，截断只是把概率压到合法区间）。两者在算法语义上根本不同，不能用同一套截断规则。
